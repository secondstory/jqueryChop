jQueryChop (v1.0.0)

jQueryChop is a simple jQuery plugin which takes an HTML img tag and converts it into a collection of arbitrarily-sized rectangles. The image manipulation is done via the canvas tag and thus a browser supporting the canvas tag or a plugin to enable the canvas in unsupported browsers is required.
